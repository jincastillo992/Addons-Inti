# Addons para la comunidad Inti Illimani

Esta compilación de addons para **World of Warcraft: Wrath of the Lich King** nace desde la hermandad Inti Illimani Histórico, pensada por y para la comunidad. Reúne herramientas seleccionadas para mejorar la experiencia de juego, optimizar el rendimiento en bandas y mazmorras, y mantener una interfaz clara sin perder la esencia del juego.

El objetivo es simple: jugar mejor, organizarnos como hermandad y enfrentar juntos los desafíos de Rasganorte, manteniendo siempre el espíritu que nos identifica.

<img src="./src/image/logo-inti.png" alt="logo-inti" width="200"/>


# Como instalar los Addons

- Descarga un Addon
- Descomprime la carpeta
- Abre la carpeta wow en `wow/Interface/Addon`
- Arrastra y suelta la(s) carpeta(s) dentro de la carpeta de addons (el nombre de la carpeta debe coincidir con el **archivo .toc** que contiene; si la carpeta se llama `"*-master"`, entra dentro de esa carpeta y copia/pega las carpetas correspondientes).
